================================================================================
HOW TO RUN
================================================================================

1. Install dependencies:
   $ cd payo-apps
   $ npm install

2. iOS setup:
   $ npm run pod-install

3. Run the app:
   $ npm run ios        # For iOS
   $ npm run android    # For Android

================================================================================
CONFIGURATION NEEDED
================================================================================

1. Update blockchain addresses in src/constants/index.ts:
   - TOKEN_ADDRESS: Your PAYO token contract
   - ORACLE_ADDRESS: Your price oracle contract

2. Update API endpoint in src/constants/index.ts:
   - BASE_URL: Your backend API URL

3. Add camera permissions (for QR scanning):
   iOS: Info.plist
   Android: AndroidManifest.xml
